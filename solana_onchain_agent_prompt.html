Heurísticas Implementadas
Common Fee Payer Se wallet A paga fee de txs originadas por B, C, D → provável controle centralizado. Timing Cluster Txs com delta < 500ms entre wallets diferentes → bot ou script controlador comum. Value-Round Pattern Valores exatos (1.000, 5.000 SOL) indicam automação — não humano.
Peeling Detection Detecte saídas decrescentes: 100 → 99.9 → 99.8 SOL (fee peel) rastreie o valor residual, não o fragmentado. Bridge Correlation Valor X sai em Solana via bridge → procure valor ~X±slippage chegando na chain destino no mesmo window temporal.
Exchange Attribution Depósito em endereço com label CEX + withdrawal em sequência = KYC choke point. Documente o par (deposit addr, time). PDA Authority Tracing Quem tem authority sobre o PDA controla o programa. Trace a upgrade authority chain completa.
Níveis de Confiança HIGH: on-chain proof direta MEDIUM: heurística + 2 sinais LOW: hipótese única, 1 sinal Nunca collapse MEDIUM→HIGH sem evidência adicional. Documente cada inferência.
